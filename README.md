# Программа для построения иерархии классов на основе правил вывода.

---

## 📌 Функциональность
- Парсинг конфигурационных JSON-файлов с извлечением описания классов и их свойств
- Автоматическое определение отношений наследования и вложенности между классами
- Семантическая и структурная валидация входных данных
- Транзитивная редукция (минимизация) построенного графа для удаления избыточных дублирующих связей
- Экспорт оптимизированной структуры иерархии классов в текстовый файл разметки Graphviz DOT

---

## ⚙️ Требования
- **Qt 6.0+/Qt 5.15+** (модули: Core)
- **C++17** или новее
- **Сборка**: QMake или CMake

---

## 🚀 Запуск

$: buildClassHierarchy.exe <входной_файл.json> <выходной_файл.dot>
- P.S. Может потребоваться указать абсолютные пути к файлам (например: D:\...\input.json D:\...\output.dot).
- P.S.2 Для работы скомпилированного консольного приложения в среде Windows требуются соответствующие .dll-файлы зависимостей библиотек Qt.

---

## 📝 Пример

### Входной JSON-файл 

* `{`
* `    "classes": [`
* `        {`
* `            "class_name": "BaseSchedule",`
* `            "properties": [`
* `                {`
* `                    "name": "lessons",`
* `                    "value_count": [5]`
* `                }`
* `            ]`
* `        },`
* `        {`
* `            "class_name": "AdvancedSchedule",`
* `            "properties": [`
* `                {`
* `                    "name": "lessons",`
* `                    "value_count": [5]`
* `                },`
* `                {`
* `                    "name": "exam_period",`
* `                    "expected_value": [1, 2, 3]`
* `                }`
* `            ]`
* `        }`
* `    ]`
* `}`

### Итоговый граф в формате DOT
* `digraph G {`
* `   "BaseSchedule" -> "AdvancedSchedule";`
* `}`
